How Link JavaScript & CSS Files
To Your HTML Document

First you need all three documents; html file, JavaScript file, and CSS file saved to the same directory.

For the CSS file, create a <link> tag within the <head> tag. The link tag is a self-closing tag. It should include three attributes in any order. First is the href and set it equal to the relative path to the CSS file. Second is the type attribute, set equal to "text/css". Lastly, the rel attribute should be equal to "stylesheet".

To link the JavaScript file, create an opening and closing <script> tag before the closing </body> tag. Then within the opening <script> tag add the following attributes. First a src attribute set equal to a relative path to the JavaScript file. Second, a type attribute set to "text/javascript".
